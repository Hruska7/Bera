# Querying via CLI
 
<Callout type="info" emoji="ℹ️">
  Stargazer CLI will work the same way as Cosmos SDK. Replace `simd` with `stargazerd`
</Callout>

Stargazer does not extend the use of CLI querying past the base implementation of the Cosmos SDK.
For information on querying via CLI in Stargazer you can reference [Using the CLI](https://docs.cosmos.network/main/run-node/interact-node#using-grpc) 
from the Cosmos SDK documentation.
