# Using Hardhat with Stargazer EVM

## Introduction

This guide shows how to deploy and interact with smart contracts using Hardhat
on a local Stargazer EVM network.

[Hardhat](https://hardhat.org/) is a development environment for Ethereum software. It consists of different components for editing, compiling, debugging and deploying your smart contracts and dApps, all of which work together to create a complete development environment.

## Recommended Knowledge

- Basic understanding of [Solidity](https://docs.soliditylang.org) and Cosmos.
- Basic understanding of the [Stargazer's architecture]() 
- Basic understanding of [Olympus DAO](https://www.olympusdao.finance/)
- Basic understanding of Javascript

## Requirements

- [NodeJS](https://nodejs.org/en) version `16.x`
- You have [Hardhat installed](https://hardhat.org/hardhat-runner/docs/getting-started#installation).
- You have a running instance of Stargazer EVM. See [Starting A Node]()
