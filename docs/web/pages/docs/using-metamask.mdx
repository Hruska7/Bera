# Using Metamask With Stargazer

In this section, we will guide you through the process of 
downloading Metamask and connecting it to a local Stargazer node. 
Before proceeding, please ensure that you have a local node up and running. 
If you need help with this, you can refer to the [Your First Node]() section for instructions.

To begin, you will need to download and install the Metamask browser extension. 
Once installed, you can then connect your Injected Metamask to your local Stargazer 
node by following the steps provided in this section.

 
<Callout type="warning" emoji="⚠️">
  Take note of your private key and keep it somewhere safe. click [here](https://learn.metamask.io/lessons/what-is-a-self-custody-wallet) to learn about self custody.
</Callout>

## Downloading And Installing Metamask
You can download and install the Metamask browser extension [here](https://metamask.io/download/).

## Adding Stargazer Local Network
Click to MetaMask icon on the browser and select the network drop-down menu. Here we should connect to C-Chain. Click to "Custom RPC".

![](/metamask-add-network.png)

Remove the existing `Localhost 8545` default network.

Now, we need to input the following values/

- Network Name: `Stargazer`
- New RPC URL: `http://localhost:8545`
- Chain ID: `stargazer420_69-1`
- Symbol(optional): `GAZER`
- Block Explorer URL (optional): ``

Click `Save`

The Stargazer local environment is now available on your local machine.
